this. this(0)의 차이

this. :
	현재 객체를 가리키는 특수한 키워드
	인스턴스 변수와 메서드의 매개변수 또는 지역변수의 이름이 같을  혼동을 피하기 위해
	예제 코드 : this.field 현재 객체의 myfield 인스턴스 변수를 가리킴
this(0) :
	현재 클래스의 다른 생성자를 호출하는 방법
	주로 생성자 오버로딩 시 사용되며, 한 생성자에서 다른 생성자를 호출하는데 사용
	this(0) 인자로 0을 가진 다른 생성자를 호출하는 것
	생성자 간의 호출을 통해 코드 재사용성을 높이고 중복을 피할 수 있음
